Tea Time In Space

Tea Time In Space is a chrome extension that replaces the default new tab page.

It leverages external APIs to display interesting facts about space, such as NASA's astronomy photo of the day, the current coordinates of the International Space Station and more. Click one of the links below to check it out for yourself!

WorkLand

WorkLand is a retro-themed, interactive project management application that allows users to manage projects and tasks with a moveable avatar. It leverages web sockets to allow for real-time collaboration between users.

This was a group project that I worked on with 2 other developers for my final project at Lighthouse Labs. Unfortunately, the app is no longer live, but you can check out the code on Github!
